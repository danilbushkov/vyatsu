Компиляция:
    fpc arena.pas
Компиляция ботов:
    - Ботов нужно расположить по файлам ubot1.pas и ubot2.pas
    - Компиляция библиотек ботов: fpc libbot1.pas и fpc libbot2.pas 
    
Запуск:
    - В терминале arena.exe
    - После запуска арены пользователью нужно ввести количество сетов

Возможные баги:
    - Если имеется массив с картами, то при передаче из getMap элемента массива, возращается 
    ссылка на массив и поэтому значения карты могут поменяться. После сета желательно инициализировать новую 
    карту
